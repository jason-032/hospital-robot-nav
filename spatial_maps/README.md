This ROS2 package provides complete navigation capabilities for indoor robots using:
- **Occupancy Grid Maps** - For collision avoidance and path planning
- **Topological Network** - For semantic navigation and efficient pathfinding
- **Gazebo Simulation** - For testing and validation

## Quick Start

### 1. Build Package
```bash
cd ~/ros2_ws
colcon build --packages-select spatial_maps
source install/setup.bash
```

### 2. Launch Simulation
```bash
ros2 launch spatial_maps multi_robot_simulation.launch.py
```

### 3. Send Navigation Goal
```bash
# Semantic goal (room name)
ros2 topic pub /robot_1/goal_location std_msgs/String "data: 'Room 001'"

# Pose goal
ros2 topic pub /robot_1/goal_pose geometry_msgs/PoseStamped "{header: {frame_id: 'map'}, pose: {position: {x: 10.0, y: 5.0, z: 0.0}}}"
```
